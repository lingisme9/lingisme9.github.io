html学习笔记

一、实体

1.使用实体的原因：多个空格只认一个空格只认一个
        2.在html中不能书写特殊符号：空格，大于小于
        3.如果需要书写特殊符号那么就需要特殊符号：实体
        4.实体的语法
            &实体的没名字
                &nbsp；：表示空格
                &gt；：大于号
                &lt：小于号
                &copy；：版权符号
        5.怎样查询实体：w3school实体

二、meta标签

    1.meta主要用于表示元数据，元数据不给用户看
    2.charset 指定网页字符集
    3.name 指定数据名字
    4.content 指定数据内容
        keywords:表示网页关键字，可以设置多个关键字，用“，”隔开
        description：网站描述会显示在搜索描述里面
    5.title标签的内容：超链接上的标签显示
  *********************将页面重定义到另一个网站

三、语义化标签

网页中html负责网页结构，
在使用时应该关注语义而不是样式

        标题标签
            h1~h6
            h1只有一个
            一般情况下标题只用到h1~h3

        块元素：在页面中独占一行block element
            hgtoup:用来给标签分组
           <hgroup>
                    <h1>一级标题</h1>
                    <h2>二级标题</h2>
                 </hgroup>

        em标签：用于语音语调加重 
                 行内元素：不会独占一行:inline element
            <!-- strong :表示强调重要内容-->
            <p>你今天不需要<strong>完成作业</strong></p>



      blockquote:表示长引用，换行
      鲁迅说：
      <blockquote>
          这句话我没说过
      </blockquote>
      <!-- 不换行：行标签 -->
      鲁迅说：
      <q>
          这句话我没说过
      </q>
      <!-- br -->
      <br>
      这句话我没说过
      <!-- 乱码处理，点击utf-8 -->


    块元素 block element
        --对页面进行布局，一块一块的
    行内元素 inline element
        --行内元素主要用来包裹文字
        --一般情况下在块元素中放行内元素
        --块元素中基本什么都能放
        --p元素中不能放任何块元素
    浏览器在解析网站时会修正不符合规范的内容
    比如：
        -标签写在外部
        -p元素中嵌套了块元素
        -根元素中出现了除了head，body以外的子元素
    浏览器开发工具：element右键查看：内存中的代码存在

 <!-- 
    布局标签（结构化语义）
    header:头部
    main:主体,一个页面只有一个
    footer:尾部
    nav:导航：一堆分类那块
    aside：主体相关的不属于主体的内容
    article：独立文章
    section：独立区块：上面的标签都不能表示时

    *div：没有语义，表示区块：目前来讲的主要布局元素
    *span:行内元素,没有语义，主要用于选中文字
 -->

四、列表

 <!-- 
    列表:list
        -铅笔
        -尺子
        -橡皮
    在html中也可以创建列表：
        -有序列表
        -无序列表
        -定义列表
        ul：无序列表
            -使用li来表示列表项
        ol：有序列表
            -li
        dl：定义序列
            dt来表示定义内容
            dd来解释说明
        列表之间可以互相嵌套
            ul中套ul
 -->

五、超链接

超链接：
a标签：（行内标签）
    -属性：href：指定跳转路径
        -值可以是外部网站地址
        -可以是内部页面地址：同一目录下就可以
超链接a标签是行内标签，在a标签中可以嵌套除它自生外的任何元素

六、相对路径

    跳转服务器内部页面时要相对路径：
       ./或../开头
       可以省略
       ./:表示当前文件所在目录
            09.相对路径.html
            在同一目录下可以不写
       ../：上一级目录中查找
       读当前，读下一级，读上一级;

七、超链接的其他应用

 a标签的属性值：
        target:指定超链接打开的位置:新标签
            -_self 默认值:当前标签页
            -_blank 新的页面中打开标签页

 <a href="#bottom">去底部</a>

 <a href="09.相对路径.html" target="_blank">超链接</a> 
 回到顶部：
        id属性:唯一不重复的
            -每一个标签都可以添加id属性,唯一标识,同一个页面中不能相同出现id属性
            -字母开头
        可以跳转到指定位置:#标签

 在开发中可以用：
        -#去代表一个占位符会刷新页面
        -javascript:;占位符,不会发生任何反应

八、图片标签

    图片标签:当前页面引入图片
        img:自结束标签
            属性
                src：指定路径
                img属于替换元素：img所引用的外部元素替换img
                alt：图片属性,默认不会显示，无法加载时可能会显示
                搜索引擎会按alt来识别图片
                width：宽度
                hight：高度
                只修改一个：等比缩放
                通常指定一个就行
            注意：不建议修改图片大小
                 移动端经常修改
    图片的格式：
        jpg:颜色多，不支持透明，不支持动图
            -表示照片
        gif:支持颜色少，支持简单透明，支持动图
            -颜色单一的图片，动图
        png:支持颜色丰富，支持复杂透明，不支持动图
            -颜色丰富，复杂透明
        webp:
            谷歌新推出网页的图片格式
            具备其他格式的所有有点，文件还小
            效果一样用小的，效果不同用好的
            兼容不好

也可以复制图片代码直接等号后面 
      base64
        -将图片用base64编码，这样可以将字符转换为字符，通过字符形式引入图片
        -一般都是和网页一起加载的图片才会使用，不会出现延迟，加载速度快

九、内敛样式

    iframe
    内联框架：用于当前页面中引入别的网站
        -src：网页路径
        -frameborder：指定内联框架的边框：不好看
            -会产生凹下去的外框，不好看
                -1：打开
                -0：关闭

十、音视频

    audio:像页面中引入音频文件
    属性：
        -controls：是否允许用户控制
        -autoplay：自动播放
            -如果设置，打开页面时会自动播放，目前大部分浏览器不会自动播放
            -先播放一次之后就可以自动播放了，IE可以
        loop:循环播放
    除了scr指定文件之外;还可以用scorce来指定文件
        -指定多个文件，不会全部执行，找一个支持的
    embed:老版本的播放，自动播放，第二次会变很小：指定宽高
        -type：指定类型




    video：引入视频文件
    属性：
        -autoplay
        -loop
        -音频通常用source，这样兼容性好

 一般不用这种绝对路径的方法
         allowFullScreen="true"：全屏
         一般引入：和之前一样：
